\doxysubsection{Detailed Description}
Check if a type has a posterior function or not. 

A \mbox{\hyperlink{class_top_n}{TopN}} is a n object you can \char`\"{}add\char`\"{} hypotheses to (via add or $<$$<$) and it stores the best N of them. This is widely used in Fleet in order to find good approximations to the top hypotheses found in M\+C\+TS or M\+C\+MC. The default constructor here reads from \mbox{\hyperlink{namespace_fleet_args_acbc0f011abb2f878bf68a85617d8c183}{Fleet\+Args\+::ntop}} as the default value.

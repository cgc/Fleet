Detailed Description

This is needed below so we can check if something is a Builtin without template args.

These are convenient wrappers for built-in operations that just bundle toegether format, op, and p this is useful because then we can put the builtins in the same PRIMITIVES bundle so for instance we can have Primitives::If<S>("if(%s,%s,%s)", 1.0) in PRIMITIVES, even though If is a built-in. So these are just data structures that hold this information until grammar.add, where they are mapped onto Rules for builtins.
